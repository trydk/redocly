# Redocly Docker

## Installation

Please prepare a `./swagger/swagger.json` to be generated by `docker-compose`

## Run

    $ ./start.sh

### Manual Run

    $ docker-compose up --build -d

## Open Browser

Serve local file from `./swagger/swagger.json` and watch for updates

    http://localhost:8080
